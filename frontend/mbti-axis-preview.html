📊 ABSA 점수 계산 방식

S_score = 매운맛 + 짠맛 + 강렬한맛

M_score = 담백함 + 단맛 + 부드러운맛

A2_score (축2) = 공간 + 분위기 + 웨이팅 + 쾌적함 + 인테리어

O2_score (축2) = 주차 + (1 - 웨이팅) + 접근성 + 빠른서비스

P_score = 품질/신선도 + 맛 + 재료의질

C_score = 가격(if > 0) + 양 + 가성비

A4_score (축4) = 공간 + 분위기 + 서비스 + 함께식사하기좋음

O4_score (축4) = (1 - 소음) + 쾌적함 + 혼자식사하기좋음
